## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/my-react-app.git
```

Navigate into the project:

```bash
cd my-react-app
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
